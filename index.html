General Margin

The main margin of Instagram posts is 8% of the smallest side of the artboard. Images, text and graphics are placed inside this area.

Gessa Frame

If type is the only graphic element in the post, you can use GESSA Main frame. The types are placed in the corner of the frame and the background color of the artbord will be selected based on the GESSA color mode described in the color section.
